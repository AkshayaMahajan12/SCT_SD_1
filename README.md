# 🌡️ Temperature Converter in Python

A simple menu-driven Python application that converts temperatures between **Celsius (°C), Fahrenheit (°F), and Kelvin (K)**.

## 📌 Features

* Convert Celsius to Fahrenheit
* Convert Celsius to Kelvin
* Convert Fahrenheit to Celsius
* Convert Fahrenheit to Kelvin
* Convert Kelvin to Celsius
* Convert Kelvin to Fahrenheit
* Convert one temperature to both other scales
* Menu-driven interface
* Input validation using exception handling
* Repeat conversions until the user exits

## 🛠️ Technologies Used

* Python 3

## ▶️ How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/temperature-converter.git
   ```

2. Navigate to the project folder:

   ```bash
   cd temperature-converter
   ```

3. Run the program:

   ```bash
   python temperature_converter.py
   ```
